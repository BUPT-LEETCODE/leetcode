# BUPT-LEETCODE

## 提交格式

* 文件名：题目编号-题目名(空格使用'-'分割).md
* 文件格式：使用MARKDOWN格式
* 文件内容：
	* 文件头使用YAML格式，模板见本页末尾。
	* 标题：使用二级标题
	* 简介：使用英文简介
	* 解题思路：介绍思路
	* 源代码：注明代码语言

## 其他
* 内容之间隔一个空行。
* 有多种解法的可以都列出来。

暂时就想到这么多吧。
